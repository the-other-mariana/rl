Instalación de Python:
NOTA: TensorFlow solo funciona con python 3.8

 0. Instalar Python seleccionando la casilla que pide agregar la ruta de python al PATH
 1. Verificar que las siguientes 2 rutas esten en la variable PATH de windows:
      C:\Users\José Abdón\AppData\Local\Programs\Python\Python38\Scripts\
      C:\Users\José Abdón\AppData\Local\Programs\Python\Python38\
 2. Ruta de Python=C:\Users\José Abdón\AppData\Local\Programs\Python\Python38
 3. WINDOWS> python --version ..... muestra la versión de Python
 4. WINDOWS> pip --version ........ muestra la versión de pip
 5. WINDOWS> pip .................. muestra la ayuda de pip
 6. Instalar Spyder
    Configuracion: Cambiar el interprete de python
      1. Herramientas->Preferencias
      2. Interprete de Python->Usar el siguiente interprete: C:\Users\José Abdón\AppData\Local\Programs\Python\Python39\python.exe

 7. WINDOWS> pip install tensorflow
 8. WINDOWS> pip install numpy
 9. WINDOWS> pip install scipy 
10. WINDOWS> pip install matplotlib
11. WINDOWS> pip install pandas
12. WINDOWS> pip install scikit-learn

14. WINDOWS> pip install opencv-python
